# NEAT with Flappy Bird (CPP)

# Requirements

- SDL
- SDL Image
- C++ Compiler (MingW)
- CMAKE

# For Linux Installs

```
sudo apt install libsdl2-2.0.0 libsdl2-dev libsdl2-image-2.0-0 libsdl2-image-dev cmake
```

# To Run

**Linux Machine**

```
# Build Command
sh build.sh

# Run Command
sh run.sh
```
